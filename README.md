# snow-light-place

A plugin marketplace for Claude Code.

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **scribe** | Document writing workflow: draft outlines, write in parallel, export to PDF/DOCX/XLSX/PPTX/.pen/Confluence |

## Installation

```bash
# Add marketplace
claude plugin marketplace add SnowLightPath/snow-light-place

# Install plugin
claude plugin install scribe@snow-light-place
```
